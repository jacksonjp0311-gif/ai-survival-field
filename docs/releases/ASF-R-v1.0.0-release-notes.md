# ASF-R v1.0.0 Release Notes

ASF-R v1.0.0 is the first bounded governed release.

It does not expand authority.

It freezes the existing loop as installable, testable, documented, and remotely
evidenced.

## Public Claim

ASF-R is a local evidence-gated control loop for AI-generated software actions.
It checks, blocks, wounds, repairs under authorization, validates closure, and
preserves evidence before action becomes durable state.
