# v0.3 Dry-Run Boundary

ASF-R v0.3 proves adapter dry-run enforcement only.

Core law:

```text
Dry-run may reveal the shape of a repair or mutation.
Dry-run may not perform it.
```

Forbidden in v0.3:

- live mutation,
- live self-healing,
- adapter self-authorization,
- treating dry-run success as production permission,
- closing wounds from dry-run alone.
