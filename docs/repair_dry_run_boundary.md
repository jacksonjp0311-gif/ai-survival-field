# Repair Dry-Run Boundary

Repair dry-run may reveal the shape of repair.

It may not:

- write files,
- edit repositories,
- update memory,
- mutate policies,
- close wounds,
- enable `enforce_full`,
- grant authority,
- call external mutation APIs.

Dry-run law:

```text
Dry-run may reveal the shape of repair.
Dry-run may not perform repair.
```
