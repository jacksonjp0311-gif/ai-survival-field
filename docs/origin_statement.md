AI Survival Field Runtime is the production-oriented successor line to Survivor
Field Theory v1.4.

SFT v1.4 remains the reference theory and experimental validator.

ASF-R is the production runtime line:

```text
SFT v1.4
  -> reference validator
  -> negative-power theory
  -> bounded experimental artifact

ASF-R
  -> rehydration
  -> RCC Nexus routing
  -> policy-as-code
  -> wound packages
  -> evidence ledger
  -> adapters
  -> geometric operator UI
```

The successor relationship is conceptual, not a mutation of the reference repo.
