# Repository Hygiene Guard

Governance surfaces must remain parseable.

Limits:

```text
warning_limit_bytes: 10000000
hard_limit_bytes: 95000000
```

Repository hygiene law:

```text
Governance surfaces must remain parseable.
If the signal becomes too large to ingest, it becomes operational noise.
```

The hygiene guard is not a production quality claim. It is a repository
continuity guard.
