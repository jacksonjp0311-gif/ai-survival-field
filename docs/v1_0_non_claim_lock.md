# v1.0 Non-Claim Lock

ASF-R v1.0-rc1 is a bounded production release candidate.

It does not prove truth.

It does not make AI safe.

It does not replace human judgment.

It does not provide formal verification.

It does not provide production security by itself.

It does not grant autonomous repair authority.

It does not enable `enforce_full`.

It governs continuation under explicit artifacts, policies, evidence, operator
authorization, and non-claim locks.

```text
v1.0 does not expand authority.
v1.0 makes the existing loop installable, testable, documented, and reproducible.
```
